# Multi-WeChat
Opening Multi Wechat

## Method 01
1. Hold down Enter key and double click on wechat icon
2. Hold down Shift key and double click on wechat icon

## Method 02
1. Open notepad or any other text editor
2. Paste the following code

@echo off
start "" "C:\Program Files\Tencent\Weixin\Weixin.exe"
start "" "C:\Program Files\Tencent\Weixin\Weixin.exe"
exit

Note: Path can be different 

3. save it as WeChat.bat
4. Make sure wechat application is exited
5. Open WeChat.bat file
